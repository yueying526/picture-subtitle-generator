# Movie Subtitle Image Generator 图片字幕生成器

**Create cinematic subtitle images from any photo — perfect for social media sharing.**

上传图片 + 输入台词，一键生成电影感字幕长图，适合小红书/朋友圈分享。

**Live Demo:** [picture-subtitle-generator.vercel.app](https://picture-subtitle-generator-blue.vercel.app)

## Features

- Upload any image as background
- Multi-line subtitle input — each line becomes a subtitle strip
- Customizable: font size, color, outline color, subtitle height, stroke width, bottom margin
- Real-time preview
- One-click download as PNG
- Pure frontend — no server needed, works offline

## How It Works

1. Upload a photo
2. Type your subtitles (one line per subtitle)
3. Adjust font size, color, outline
4. Click "Generate" — the tool stitches subtitle strips using the image's bottom edge as background
5. Download your cinematic long image

## Use Cases

- Movie quote sharing on Xiaohongshu / Instagram
- Song lyric images
- Book quote cards
- Meme creation with custom subtitles

## Tech Stack

- Pure HTML5 + CSS3 + JavaScript
- Canvas API for image processing
- Zero dependencies, runs in any modern browser

## Run Locally

Just open `index.html` in your browser. No build step needed.

Or deploy to Vercel/Netlify with zero config.
